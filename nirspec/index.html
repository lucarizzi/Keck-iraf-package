The NIRSPEC IRAF Reduction Package

This package contains the following IRAF tasks related to reductions of low-dispersion spectra from the NIRSPEC instrument at the W. M. Keck Observatory.

Tasks

The following tasks are available in this package:
mktracer
Create a trace image from NIRSPEC low-dispersion flat
xdistcor
Remove x distortion from NIRSPEC image
ydistcor
Remove y distortion from NIRSPEC image
plotcuts
Overplot cuts from multiple places along a spectrum
spec_s2n
Estimate the S/N for multispec-format spectra
skyplot
Create simulated sky spectrum and corresponding line list
skyinterp
Interpolate over spectral regions affected by strong sky lines
donspec
Rectify, extract, and calibrate NIRSPEC low-d spectra

Installation

  • Download the package installation instructions
  • Download the current version of the package from the Keck FTP server as a gzip-ped tarfile.
